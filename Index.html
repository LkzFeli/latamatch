🔥 CONFIGURAR FIREBASE

Para que la app funcione online con usuarios reales de todo LATAM necesitás Firebase (gratis hasta ~50k usuarios).

1. Ir a console.firebase.google.com
2. Crear proyecto → darle nombre "latamatch"
3. Agregar app Web (</>)
4. Copiar el objeto firebaseConfig
5. Pegarlo en el código del HTML (arriba del todo)
6. Ir a Authentication → Email/Password → Habilitar
7. Ir a Firestore Database → Crear base
8. Copiar las reglas de seguridad del comentario del código
9. (Opcional) Storage para fotos de perfil

También podés contratar a alguien para que lo configure por vos, o preguntarme más detalles.
